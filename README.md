# ProteoAnalystR

`ProteoAnalystR` contains the R-side analysis and visualization functions used by the ProteoAnalyst web platform.

## Notes For Developers

- This directory is used as application runtime code, not as a standalone CRAN-style package.
- Keep documentation synchronized with the JSF pages and Java calls in the main ProteoAnalyst app.
- If plot behavior changes, update both the relevant R script comments and this README so the embedded package description stays accurate.
